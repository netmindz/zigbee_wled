What you get

  • Hue-compatible Zigbee endpoints — each light appears in the Hue app
  • Wired DMX512 or ArtNet output, selectable from the web UI
  • Up to 16 RGB / RGBW lights with per-light DMX address mapping
  • Web configuration UI with captive portal for WiFi setup
  • CIE XY, Hue/Sat, and Color Temperature support

After flashing

  1. Connect to the ZigbeeDMX-Setup WiFi network
  2. Configure your home WiFi in the captive portal
  3. Open the device's IP in a browser to configure lights & output
  4. Pair with your Hue Bridge via Settings → Lights → Search
